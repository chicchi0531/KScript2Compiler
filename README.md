
# Kscript2とは
Kscript2はCライクな手続き型言語と、ノベルスクリプトを合体させたスクリプト言語です。
この言語は、例えばRPGのような、ロジックを多く含むゲームにおける、シナリオの量産を助ける目的で設計されました。
特に、シナリオライターがある程度スクリプトやプログラムの知識を身に着けているようなインディーゲーム開発において、本言語は最大の力を発揮できるように作られています。

一般のノベルスクリプト言語に比べ、ロジックが組みやすい代わりに、基本的なC系言語の知識を必要とし、非プログラマには導入コストが高い作りとなっています。

その代わり、一旦習得してしまえば、少々複雑な条件分岐などのロジックを、従来のDSLやMarkupLangageライクなノベルスクリプト言語と比較して、極めて明快に、読みやすく書くことができます。

本言語はシナリオ自体の書きやすさにも重点をおいており、
例えば従来の言語では、名前を表示して、その後セリフを表示するようなスクリプトを、

```
[showname str="キャラ名"]
こんにちは。おげんきですか？[p]
```
のように記述していたところを、
下記のような形で書くことができます。

```
- キャラ名
こんにちは。おげんきですか？」
```

[]のような、半角切り替え操作を減らし、改ページについてもタグではなく、小説などで一般的な"」"記号で代用できるようにすることで、シナリオライターが規定の書式に沿って書いたシナリオを、そのままスクリプトとして使えるようになっています。
これは、シナリオライティングとスクリプティングの作業が、極めてシームレスに行えることを指します。

## 動作環境
### コンパイラ
Windows10 x86_64
MacOS 11.1 M1以降

### 仮想マシン（設計中）
Unity 2020.1以降

## 機能
- 関数
- 基本的な制御文（if-else）
- 基本的なループ分（for, while）
- 基本的な演算（+ - * / % || && <= >= < > ==）
- ノベル構文
- c#の関数呼び出し
